## 🛠️ Tech Stack & Architecture

This landing page is engineered purely with native web technologies to ensure zero dependency overhead and sub-millisecond load times.

* **HTML5**: Semantic layout optimized for accessibility (`aria-labels`) and structural indexing.
* **CSS3 (Custom Variables)**: Uses atomic design tokens for easily tweakable styling, glassmorphism filtering (`backdrop-filter`), and keyframe pulse animations.
* **Vanilla JavaScript (ES6+)**: Handles high-performance 2D Canvas rendering, coordinate-mapping, and programmatic time conversions.
